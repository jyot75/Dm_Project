What is graph ?

Graph is a set of vertices(or nodes) and a set of edges which connects a pair of nodes. In mathematics, graph theory is the study of graphs, which are mathematical structures used to model pairwise relations between objects. A graph in this context is made up of vertices (also called nodes or points) which are connected by edges (also called links or lines).

What is graph coloring ?

Graph coloring is the assignment of different colors ( or labels ) to the vertices of a graph, such that no 2 adjacent (connected) vertices have the same color.


Procedure of sudoku solving

Lets understand how graph coloring method is useful in solving sudoku. We try to understand this method via solution of 4*4 sudoku. So we have 16 cells and we represent each cell via 1 to 16 digits as shown in below figure.

Let we take 4 different color like red, green, blue and yellow in our 4x4 sudoku, So we can put any number from 1 to 4 in each cell. So we can distribute each color to 1 to 4 digit. First let we assign red color to digit 1,assign green color to digit 2,assign blue color to digit 3 and assign yellow color to digit 4.

Rules for to generate edges

To generate edges, Start with vertice that represent to first cell.

  1. Now connect edges between this selected vertice and its all row's vertices and its all column's vertices and its all 2*2 subbox's vertices.
  2. If selected cell is filled with any digit and if any other cell that connected via edges with that vertice has same colour as selected vertice then solution of this sudoku is not possible, but there is no same colour with connected vertices via edges then selected vertice is safe. Then move to the next filled cell and start from the first statement. If all filled vertices are safe then move to the first empty cell and start with third statement.
  3. If selected cell is empty then connect edges like mentioned in first state- ment. Now assign colour to that selected vertice using graph coloring method. If non of the colour is selected for that vertice then move to the previous empty cell and change the colour of that previous vertice and check the condition men- tioned in third statement. (This process works recursively).

Algorithm

  1. Create a recursive function that takes current vertex index,number of vertices and output color array as arguments.
  2. If the current vertex index is equal to number of vertices then return true and print the colour configuration in the output array.
  3. Assign colour on a vertex (1 to m).
  4. For every assigned color, check if the configuration is right not break rules, (i.e. check the adjacent vertices do not have the same colour)then recursively call the function with next index and number of vertices.
  5. If any recursive function returns true break the loop and return true.
  6. If no recursive function returns true, then return false.
